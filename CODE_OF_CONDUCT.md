# Code of Conduct

Clew is a small public project. The goal is to make it possible to discuss product ideas, bugs, docs, and contributions without turning the repository into a hostile place.

## Expected behavior

- Be direct without being cruel.
- Critique ideas, code, and decisions instead of attacking people.
- Assume other contributors are here to build something real.
- Leave enough context for others to understand your point.
- Respect the difference between disagreement and disrespect.

## Unacceptable behavior

- harassment or intimidation
- personal insults
- discriminatory language
- bad-faith trolling
- sharing private information without consent
- derailing technical conversations into hostility

## Project tone

This project values strong opinions, clear technical critique, and honest product discussion. That is welcome. What is not welcome is turning intensity into contempt for other people.
